# Solana Getting Started

Learn how to build Solana indexers using Squid SDK. Index transactions, instructions, and account changes on the Solana blockchain.

## Prerequisites

- Node.js v18+
- Basic understanding of Solana blockchain
- Familiarity with TypeScript

## Quick Start

<Steps>
  <Step title="Install Squid CLI">
    ```bash
    npm i -g @subsquid/cli
    ```
  </Step>
  
  <Step title="Create Solana Project">
    ```bash
    sqd init solana-indexer -t solana
    cd solana-indexer
    ```
  </Step>
  
  <Step title="Install Dependencies">
    ```bash
    npm install
    ```
  </Step>
  
  <Step title="Configure Processor">
    Edit `src/main.ts` to configure your Solana data sources and processing logic.
  </Step>
</Steps>

## Basic Solana Processor

```typescript
import { SolanaBatchProcessor } from '@subsquid/solana-processor'
import { TypeormDatabase } from '@subsquid/typeorm-store'

const processor = new SolanaBatchProcessor()
  .setGateway('https://v2.archive.subsquid.io/network/solana-mainnet')
  .setRpcEndpoint('https://api.mainnet-beta.solana.com')
  .addInstruction({
    programId: ['11111111111111111111111111111112'], // System Program
    isCommitted: true,
  })
  .setFields({
    instruction: {
      programId: true,
      accounts: true,
      data: true,
    },
    transaction: {
      signatures: true,
    },
    block: {
      timestamp: true,
    },
  })

const db = new TypeormDatabase()

processor.run(db, async (ctx) => {
  for (let block of ctx.blocks) {
    for (let instruction of block.instructions) {
      // Process Solana instructions
      console.log(`Instruction: ${instruction.programId}`)
    }
  }
})
```

## Key Concepts

### **Instructions vs Transactions**
- **Instructions**: Individual operations within a transaction
- **Transactions**: Atomic units containing one or more instructions
- **Programs**: Smart contracts on Solana

### **Account Model**
- Everything is an account on Solana
- Accounts can store data or executable code
- Track account changes and ownership transfers

### **Data Sources**
- **SQD Network**: Pre-filtered Solana data for fast sync
- **RPC Endpoints**: Direct access to Solana nodes
- **Real-time Data**: Process latest transactions as they happen
