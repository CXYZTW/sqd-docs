# SQD Network Portal API

The SQD Network Portal provides REST APIs for accessing blockchain data from various networks. Our APIs are organized by blockchain type to provide optimized access patterns for different data structures.

## Available APIs

<CardGroup cols={2}>
  <Card title="EVM API" icon="ethereum" href="/en/api-reference/evm/overview">
    Access Ethereum and EVM-compatible blockchain data including transactions, logs, traces, and state diffs
  </Card>
  <Card title="Solana API" icon="solana" href="/en/api-reference/solana/overview">
    Query Solana blockchain data including instructions, transactions, balances, and rewards
  </Card>
</CardGroup>

## Getting Started

All APIs follow RESTful conventions and return JSON responses. Key features include:

- **Real-time streaming**: Get live blockchain data as it's produced
- **Historical data**: Access complete blockchain history from genesis
- **Flexible filtering**: Query specific transactions, accounts, or contract interactions
- **Rate limiting**: Built-in protection with clear error responses

## Authentication

Currently, the SQD Network Portal APIs are publicly accessible without authentication. Rate limits apply:

<Warning>
  All endpoints are rate limited at **50 requests per 10 seconds per IP**. Higher bandwidths will be available via the public network soon.
</Warning>

## Base URLs

Each blockchain network has its own dedicated endpoint:

- **Ethereum Mainnet**: `https://portal.sqd.dev/datasets/ethereum-mainnet`
- **Solana Mainnet**: `https://portal.sqd.dev/datasets/solana-mainnet`
- **Other networks**: See [Supported Networks](/en/supported-networks) for complete list

## Common Response Codes

| Code | Description |
|------|-------------|
| `200` | Success - Returns requested data |
| `204` | No content - Requested range is above available data |
| `400` | Bad request - Invalid query or parameters |
| `404` | Dataset not found |
| `409` | Conflict - Mismatched parent block hash |
| `429` | Rate limit exceeded |
| `500` | Internal server error |
| `503` | Service temporarily unavailable |
